## Prompt Notes

### Option 1

Native macOS desktop app concept, dark graphite theme, SwiftUI-style native macOS window, full-height left sidebar, large center workspace, slim right inspector, bottom chat composer, collapsible bottom console drawer, parquet data explorer and DuckDB query results visible, subtle separators, Apple-like refinement, SF Symbols, quiet premium developer tool aesthetic, not a web app, 16:10 screenshot, high fidelity product design mockup.

### Option 2

Native macOS research workbench app concept, dark graphite theme, left sidebar, center analysis canvas with stacked cards for SQL, tables, charts, notes, slim right inspector, bottom chat composer, native macOS controls and toolbar, premium Apple-style desktop application, designed for parquet and DuckDB analysis, quiet materials, high fidelity screenshot, not a browser app, 16:10.

### Option 3

Native macOS assistant app concept, closest to a premium operator console, dark graphite theme, full-height left sidebar, large chat transcript center stage, inline result cards, bottom message composer, optional subtle terminal drawer, native toolbar and sidebar behavior, Apple desktop app feel, minimal chrome, high fidelity screenshot, not a web app, 16:10.
